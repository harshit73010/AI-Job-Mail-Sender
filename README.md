# 📧 AI Job Mail Sender

AI Job Mail Sender is a Streamlit-based application that helps job seekers send professional job application emails with resume attachments directly through Gmail SMTP.

## Features

* Send job application emails to recruiters
* Upload and attach PDF resumes
* Multiple job role templates
* Customizable email subject and body
* Gmail SMTP integration
* Simple and user-friendly Streamlit interface

## Tech Stack

* Python
* Streamlit
* SMTP (Gmail)
* Email MIME Libraries

## Installation

```bash
git clone https://github.com/yourusername/AI-Job-Mail-Sender.git
cd AI-Job-Mail-Sender
pip install -r requirements.txt
```

## Run Application

```bash
streamlit run app.py
```

## Requirements

* Python 3.10+
* Gmail Account
* Gmail App Password

## Usage

1. Enter your Gmail address.
2. Enter your Gmail App Password.
3. Enter the recruiter's email.
4. Select the job role.
5. Upload your resume PDF.
6. Click **Send Email**.
